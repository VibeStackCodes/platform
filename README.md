# VibeStack Platform

AI-powered app builder — describe an app in natural language and get a full Vite + Supabase project with live preview.

## Quick Start

```bash
bun install
bun run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the client.

## Stack

Vite 8 SPA + Hono API server, React 19, TanStack Router, Tailwind CSS v4, shadcn/ui, Drizzle ORM, Supabase, Mastra agent framework, Stripe, Daytona sandboxes.

See `CLAUDE.md` for full architecture documentation.
