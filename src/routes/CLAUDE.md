# Routes — TanStack Router (File-Based)

File-based routing via TanStack Router. Route tree auto-generated in `routeTree.gen.ts`.

## Structure
```
__root.tsx              — Root layout (TooltipProvider + Outlet + devtools in DEV)
index.tsx               — Landing page (public)
auth/login.tsx          — Sign in/up page (public)
_authenticated/
  route.tsx             — Auth guard: beforeLoad redirects to /auth/login if !isAuthenticated
  dashboard.tsx         — Project list
  project.$id.tsx       — Builder UI (chat + preview)
```

## Key Patterns
- Auth guard in `_authenticated/route.tsx` uses `beforeLoad` — all nested routes inherit automatically
- `$id` is a dynamic param — accessed via `useParams({ from: '/_authenticated/project/$id' })`
- Route tree generated by TanStackRouterVite plugin — do NOT edit `routeTree.gen.ts` manually
- `_authenticated/route.tsx` must ALWAYS exist — all entity routes nest under it

## Gotchas
- `createFileRoute` paths must match filesystem structure exactly
- Don't remove `_authenticated/route.tsx` even conditionally — breaks all nested routes
- TanStackRouterVite() generates `as any` casts — we generate routeTree.gen.ts deterministically for sandboxed apps
