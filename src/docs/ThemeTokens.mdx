# Theme Tokens

VibeStack uses Tailwind v4's CSS-first configuration with oklch color tokens.

## Color Tokens

| Token | CSS Variable | Usage |
|-------|-------------|-------|
| `background` | `--background` | Page backgrounds |
| `foreground` | `--foreground` | Default text |
| `primary` | `--primary` | Primary actions, links |
| `secondary` | `--secondary` | Secondary UI elements |
| `muted` | `--muted` | Muted backgrounds |
| `accent` | `--accent` | Hover states, highlights |
| `destructive` | `--destructive` | Delete, error states |
| `card` | `--card` | Card backgrounds |
| `popover` | `--popover` | Popover/dropdown backgrounds |
| `border` | `--border` | Border color |
| `input` | `--input` | Input borders |
| `ring` | `--ring` | Focus ring |

## Sidebar Tokens

| Token | CSS Variable |
|-------|-------------|
| `sidebar` | `--sidebar` |
| `sidebar-foreground` | `--sidebar-foreground` |
| `sidebar-primary` | `--sidebar-primary` |
| `sidebar-accent` | `--sidebar-accent` |
| `sidebar-border` | `--sidebar-border` |

## Chart Tokens

`--chart-1` through `--chart-5` for data visualization.

## Typography

| Font | Variable | Usage |
|------|----------|-------|
| DM Sans | `--font-sans` | Body text, UI labels |
| DM Serif Display | `--font-display` | Landing page headings |
| JetBrains Mono | `--font-mono` | Code blocks, terminal |

## Border Radius

Derived from `--radius` base value:

| Token | Calculation |
|-------|------------|
| `radius-sm` | `var(--radius) - 4px` |
| `radius-md` | `var(--radius) - 2px` |
| `radius-lg` | `var(--radius)` |
| `radius-xl` | `var(--radius) + 4px` |
| `radius-2xl` | `var(--radius) + 8px` |

## Dark Mode

Dark mode is toggled by adding the `dark` class to `<html>`. All tokens have dark variants defined via `@custom-variant dark (&:is(.dark *))`.
